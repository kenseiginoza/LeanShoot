# Lean & Shoot

反射神経を使ったシューティングゲームです。

- 制作期間：1か月（2025/05 ~ 2025/06)
- 制作環境：Unity

以下のリンクからzipファイルをダウンロードし、解凍して `Lean&Shoot.exe` を実行してください。
